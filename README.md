# StreamVision 🎥✨

## Overview
StreamVision is a real‑time face‑swap and overlay application built with **Decart’s Lucid 2.0 API**.  
It allows streamers to apply creative overlays during live broadcasts on platforms such as **Twitch, YouTube Live, and TikTok Live**.  
The project is focused on **entertainment and content creation**, not impersonation.

---

## Features
- Real‑time face‑swap overlays for live streaming
- Local desktop application (no cloud hosting required)
- Integration with major streaming platforms
- Designed for creative masks, characters, and artistic effects

---

## Technology Stack
- **Decart Lucid 2.0 API** – powering real‑time AI overlays
- **Python 3.11** – core application logic
- **OpenCV** – video capture and processing
- **ONNXRuntime** – model execution
- **VS Code** – development environment

---

## Use Case
StreamVision is intended for:
- Independent streamers who want creative overlays
- Entertainment and content creation
- Safe, responsible use aligned with Decart’s policies
